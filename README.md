## Configurations

- create .env.local under root folder(package.json exists)

```
REACT_APP_API_KEY=your-api-key
REACT_APP_AUTH_DOMAIN=your-app-auth-domain
...
```

- set environment variables under window powershell

```
$env:variable='value'
```

- npm run start
